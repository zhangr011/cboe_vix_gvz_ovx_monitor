# CBOE VIX（恐慌指数）、GVZ（黄金 VIX）、OVX（原油 VIX）监控
CBOE 的 VIX GVZ OVX 是反应公众恐慌程度的重要三个指标。
本项目旨在监测期货市场的波动率，以预估当前投资市场整体环境。

特别感谢：https://github.com/kostasmetaxas/vixfuturesdatabridge.git 。
